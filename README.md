# HTML Biodata

Selamat datang di repositori **HTML Biodata**! Ini adalah website portofolio atau biodata pribadi sederhana yang dibangun menggunakan HTML dan CSS murni.

## 🌟 Fitur

- **Halaman Profil Utama (`biodata.html`)**: Menampilkan informasi pribadi seperti Nama, Email, Alamat, dan Program Studi.
- **Halaman Hobi (`hobi.html`)**: Menampilkan daftar hobi atau ketertarikan pribadi.
- **Tautan Media Sosial**: Terintegrasi dengan tautan Instagram dan WhatsApp.
- **Desain Responsif Sederhana**: Tata letak yang rapi menggunakan CSS dengan navigasi sisi (sidebar).

## 🚀 Teknologi yang Digunakan

- **HTML5**: Sebagai kerangka struktur halaman website.
- **CSS3**: Untuk memberikan gaya visual (styling) agar website terlihat rapi dan menarik.
- **FontAwesome 5**: Digunakan untuk menampilkan ikon-ikon yang menarik (seperti ikon media sosial, notifikasi, dan pesan).

## 📂 Struktur File

- `biodata.html` - Halaman utama yang berisi informasi identitas dan media sosial.
- `hobi.html` - Halaman yang berisi daftar hobi.
- `style.css` - File CSS untuk mengatur tampilan seluruh halaman.
- `photo.jpg` - Foto profil yang ditampilkan di bagian sidebar.

## 💻 Cara Menjalankan

Website ini tidak memerlukan server khusus atau proses instalasi. Anda dapat membukanya secara langsung melalui web browser:

1. *Clone* repositori ini atau unduh file ke komputer Anda.
2. Buka folder proyek.
3. Klik dua kali pada file `biodata.html` untuk membukanya di browser default Anda (Chrome, Firefox, Safari, dll).
